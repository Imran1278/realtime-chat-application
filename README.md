# Real-Time Chat Application

A modern, dynamic chat application that allows users to communicate instantly. Built with PHP and AJAX to ensure a seamless experience without refreshing the page.

##  Features
- **Real-Time Messaging:** Send and receive messages instantly using AJAX.
- **User Authentication:** Secure Signup and Login system with profile image upload.
- **User Search:** Find and connect with other users within the platform.
- **Online Status:** Real-time indicators to see who is currently active.
- **Responsive Design:** Fully optimized for both desktop and mobile views.

##  Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** PHP
- **Database:** MySQL
- **Communication:** AJAX (Asynchronous JavaScript and XML)

##  Project Structure
- `php/`: Contains backend logic for signup, login, and message handling.
- `javascript/`: Handles real-time UI updates and AJAX requests.
- `chat.php`: The main interface for conversations.
- `users.php`: Dashboard showing the list of available users.
- `style.css`: Custom styling for a clean and professional look.
